# SEO Scoring Criteria - Outdoor Butler

## On-Page SEO (25%)

### Title Tag (8%)
- Present: +2
- Under 60 chars: +2
- Contains primary keyword: +2
- Contains location (when relevant): +1
- Unique across site: +1

### Meta Description (7%)
- Present: +2
- 120-155 chars: +2
- Contains keyword: +1
- Has CTA: +1
- No trailing whitespace: +1

### Focus Keyword (5%)
- Set: +2
- Relevant to content: +2
- Matches search intent: +1

### URL Structure (5%)
- Clean, descriptive: +2
- Contains keyword: +1
- No parameters: +1
- Reasonable length: +1

## Content Quality (25%)

### Word Count
| Page Type | Floor | Good | Excellent |
|-----------|-------|------|-----------|
| Homepage | 300 | 500 | 800+ |
| Service page | 400 | 800 | 1200+ |
| Location page | 300 | 500 | 800+ |
| Blog post | 600 | 1000 | 1500+ |
| Business page | 300 | 600 | 1000+ |

### E-E-A-T Signals
- Author/brand attribution: +3
- Contact information present: +2
- Local knowledge demonstrated: +3
- Technical expertise shown: +2

### Readability
- Heading hierarchy (H1>H2>H3): +3
- Short paragraphs (2-4 sentences): +2
- Lists and tables used: +2
- Images with alt text: +3

## Technical (20%)

### Meta Tags (10%)
- Canonical present: +3
- Open Graph tags: +3
- No conflicting robots directives: +2
- Proper character encoding: +2

### Performance Indicators (10%)
- No encoding corruption: +3
- No broken internal links: +3
- Image optimization: +2
- Mobile-friendly structure: +2

## Schema Markup (15%)

### Presence (8%)
- Any schema present: +3
- Correct type for page: +3
- All required properties: +2

### Quality (7%)
- Valid JSON-LD: +2
- LocalBusiness for location pages: +2
- Service for service pages: +2
- Article for blog posts: +1

## GEO Readiness (15%)

### Citability (8%)
- Answer-first format: +2
- Self-contained answer blocks (134-167 words): +2
- Specific statistics/facts: +2
- Question-based headings: +2

### AI Accessibility (7%)
- AI crawlers allowed (robots.txt): +2
- Structured data for AI extraction: +2
- Brand consistency (same name everywhere): +1
- Clear entity definitions: +2
